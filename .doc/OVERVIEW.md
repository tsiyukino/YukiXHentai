# Overview
> Last updated: 2026-03-20 | Affects: all modules

YukiXHentai is a third-party ExHentai/E-Hentai desktop client built with Tauri 2 (Rust backend) + Svelte (TypeScript frontend) + SQLite. It is a local-first gallery manager and reader that syncs metadata and images from the site, stores them locally, and provides advanced browsing, searching, filtering, and reading beyond what the website offers.

Full project spec: see `.doc/SPEC.md`.
